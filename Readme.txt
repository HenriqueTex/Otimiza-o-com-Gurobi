Para instalação e Utilização do Gurobi os seguintes passos devem ser seguidos:

1 Dowload do Gurob Otimizer
https://www.gurobi.com/downloads/gurobi-optimizer-eula/

2 Dowload e Instação do Anaconda
https://www.gurobi.com/gurobi-and-anaconda-for-windows/

3 Instalação do Gurobi
 3.1 Abra o Anaconda Prompt e digite o comando
	conda config --add channels http://conda.anaconda.org/gurobi
 3.2 Digite o comando
	conda install gurobi
 3.3 Instale a licença do Gurobi 

4 Abra o arquivo "Resolução de Problemas Gurobi.ipynb" com o editor de python Jupyter que sera instalado junto com o Anaconda
